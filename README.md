<span style="color: #8192aa;"> 𝐇𝐢.

____
𝐴𝑏𝑜𝑢𝑡 𝑃𝑜𝑛𝑦 𝑇𝑜𝑤𝑛.

𝐼'𝑚 𝑜𝑓𝑡𝑒𝑛 𝑜𝑓𝑓𝑡𝑎𝑏 𝑜𝑟 𝑠𝑒𝑚𝑖/𝑎𝑓𝑘, 𝑠𝑜 𝑝𝑙𝑒𝑎𝑠𝑒 𝑢𝑠𝑒 𝑤ℎ𝑖𝑠𝑝𝑒𝑟 𝑖𝑓 𝑦𝑜𝑢 𝑤𝑎𝑛𝑡 𝑠𝑜𝑚𝑒𝑡ℎ𝑖𝑛𝑔, 𝑏𝑢𝑡 𝑏𝑒 𝑟𝑒𝑎𝑑𝑦, 𝐼 𝑐𝑎𝑛 𝑟𝑒𝑠𝑝𝑜𝑛𝑑 𝑙𝑎𝑡𝑒. 𝐴𝑠 𝐼 𝑠𝑎𝑖𝑑 𝐼 𝑑𝑜𝑛'𝑡 𝑚𝑖𝑛𝑑 𝑖𝑓 𝑦𝑜𝑢 𝑐 + ℎ 𝑤𝑖𝑡ℎ 𝑚𝑒. 𝐼'𝑚 𝑑𝑜𝑛'𝑡 𝑑𝑜 𝑖𝑡 𝑏𝑦 𝑚𝑦 𝑠𝑒𝑙𝑓 𝑢𝑛𝑡𝑖𝑙 𝐼 𝑘𝑛𝑜𝑤 𝑦𝑜𝑢, 𝑠𝑜 𝑦𝑒𝑎ℎ. 𝑃𝑙𝑒𝑎𝑠𝑒 𝑑𝑜 𝑛𝑜𝑡 𝑐𝑜𝑝𝑦 𝑜𝑟 𝑖𝑛𝑠𝑝𝑜 𝑏𝑦 𝑚𝑦 𝑠𝑘𝑖𝑛𝑠. 𝑃𝑒𝑜𝑝𝑙𝑒 𝑢𝑛𝑑𝑒𝑟 18 𝐼𝑊𝐶. 𝐼 𝑑𝑜𝑛'𝑡 𝑤𝑎𝑛𝑡 𝑎𝑛𝑦 𝑚𝑖𝑠𝑢𝑛𝑑𝑒𝑟𝑠𝑡𝑎𝑛𝑑𝑖𝑛𝑔𝑠, 𝑡ℎ𝑎𝑛𝑘𝑠.
___
𝐴𝑏𝑜𝑢𝑡 𝑚𝑒.

𝐶𝑎𝑙𝑙 𝑚𝑒 𝑌𝑎𝑛𝑓𝑖𝑥. 𝐼 𝑠𝑝𝑒𝑎𝑘 𝑟𝑢/𝑒𝑛𝑔. 𝐿𝑖𝑡𝑡𝑙𝑒 𝑠𝑜𝑡𝑠𝑖𝑎𝑙 𝑎𝑤𝑘𝑤𝑎𝑟𝑑 𝑖𝑔. 𝐴𝑙𝑠𝑜 𝐼'𝑚 𝑚𝑢𝑙𝑡𝑖𝑓𝑎𝑛𝑑𝑜𝑚 𝑎𝑟𝑡𝑖𝑠𝑡. 𝐼'𝑚 𝑛𝑜𝑡 𝑝𝑜𝑠𝑡𝑖𝑛𝑔 𝑎𝑡 𝑡ℎ𝑒 𝑚𝑜𝑚𝑒𝑛𝑡 (𝑜𝑛𝑙𝑦 𝑖𝑛 𝑇𝑒𝑙𝑒𝑔𝑟𝑎𝑚𝑚 𝑐ℎ𝑎𝑛𝑛𝑒𝑙 𝑎 𝑙𝑖𝑡𝑡𝑙𝑒), 𝑏𝑒𝑐𝑎𝑢𝑠𝑒 𝐼'𝑚 𝑠𝑡𝑟𝑢𝑔𝑔𝑙𝑖𝑛𝑔 𝑤𝑖𝑡ℎ 𝑡ℎ𝑒 𝑓𝑒𝑎𝑟 𝑜𝑓 𝑝𝑜𝑠𝑡𝑖𝑛𝑔,  𝑏𝑢𝑡 𝐼 𝑤𝑖𝑙𝑙 𝑠𝑡𝑎𝑟𝑡 𝑖𝑛 𝑡ℎ𝑒 𝑛𝑒𝑎𝑟 𝑓𝑢𝑡𝑢𝑟𝑒. 𝐼 𝑤𝑖𝑙𝑙 𝑢𝑝𝑑𝑎𝑡𝑒 𝑡ℎ𝑒 𝑖𝑛𝑓𝑜𝑟𝑚𝑎𝑡𝑖𝑜𝑛 𝑎𝑠 𝑠𝑜𝑜𝑛 𝑎𝑠 𝐼 𝑠𝑡𝑎𝑟𝑡 𝑑𝑜𝑖𝑛𝑔 𝑡ℎ𝑖𝑠.
𝑅𝑖𝑔ℎ𝑡 𝑛𝑜𝑤 𝑦𝑜𝑢 𝑐𝑎𝑛 𝑙𝑜𝑜𝑘 𝑎𝑡 𝑚𝑦 𝑑𝑟𝑎𝑤𝑖𝑛𝑔 𝑤𝑖𝑡ℎ 𝑍𝑎𝑚, 𝑊𝑒𝑚𝑚𝑏𝑢 𝑎𝑛𝑑 𝐸𝑔𝑔𝑐ℎ𝑎𝑛.
